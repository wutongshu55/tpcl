# 图像处理器 - 左黑白右红边效果

## 功能说明

- 图片**左半部分**转为黑白（灰度效果）
- 图片**右半部分**自动检测物体边缘并**加红**
将需要处理的图片命名为input.jpg保存到当前文件夹

## 编译方法

编译（把代码变成可执行程序）
打开终端，进入文件夹，执行：

gcc tpcl.c -o tpcl.exe -Wall -lm
编译成功后，会生成 tpcl.exe 文件。

第3步：运行
./tpcl.exe input.jpg output.jpg
（./tpcl.exe 输入图片名称 输出图片名称）

处理完成后，会生成 output.jpg，这就是处理后的图片。
